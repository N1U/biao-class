魔法四季
结合“二十四节气”的自然规律，以继承传统优秀文化、传播中国文化成果为设计灵感，自主研发设计出《魔法四季》这一课程，努力实现中华传统文化在学生中的创造性转化和创新性发展。
文化发掘
“澄海灯谜”是一种传统民俗文化。灯谜源于古代隐语，在清代康熙《澄海县志》有关于谜事活动的记载，其中较有特色的是宋代临安流传至今的“击鼓猜射”传统开猜。此外还有“送谜标”、“猜谜母”等民俗形式，这些传统文化体现着民族的智慧，成为人民大众喜闻乐见、雅俗共赏的传统文化艺术。
“澄海灯谜”于2008年6月入选“第二批国家级非物质文化遗产名录”。
目标意义
学生通过猜谜面等环节，对学生形成目标记忆、有意注意能力进行锻炼，使其逐渐养成条理性思维的习惯，达到有益于增长学生智力的目的。
