absolute + transform 控制子元素位置

子元素 absolute 后， 将 top / left 值设置为 50%，但是 absolute 定位基于元素左上角。
通过 transform 的 translate 属性控制坐标，translate(-50%, -50%) 即把元素的中心点从左上角移动到元素中心。
设置 transform: translate(-50%, -50%) 不需要知道子元素的 width / height 值，但是需要浏览器对 css3 的支持。
通过 margin-top/left: -50px（子元素宽高 100px) 居中兼容性更好，但是需要知道子元素的 width / height 值。
